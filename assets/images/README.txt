IMÁGENES REACTIVAS
==================

Coloca aquí UNA imagen por cada gesto, con EXACTAMENTE estos nombres:

    desparche.png      -> Puño cerrado            ✊  (Closed_Fist)
    perro_god.png      -> Pulgar arriba           👍  (Thumb_Up)
    perro_facha.png    -> Pulgar + meñique        🤙  (Call_Me, gesto propio)
    perro_silence.png  -> Índice arriba           ☝️  (Pointing_Up)

Formatos válidos: .png, .jpg, .webp, .gif (animado), etc.
Si falta algún archivo, la app NO se rompe: muestra un aviso con el nombre
que falta.

¿Quieres añadir otro gesto? Solo necesitas:
  1. (opcional) crear el gesto en web/hand_detection.js si no es de fábrica.
  2. agregar una entrada al mapa `kReactions` en lib/main.dart.
  3. poner la imagen aquí con el nombre que pusiste en el mapa.
